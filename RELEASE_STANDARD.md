# Produktvision

CAPS Studio ist kein eigener KI-Bildgenerator.

CAPS Studio ist der Produktionsleiter für hochwertige Kinderbuchprojekte.

ChatGPT erzeugt die Illustrationen. CAPS übernimmt:

- Projektplanung
- Szenenregie
- Prompt-Erstellung
- Bildverwaltung
- Qualitätskontrolle
- Fortschrittsverwaltung
- PDF-Projektbuch
