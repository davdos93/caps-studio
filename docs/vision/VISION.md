# Produktvision

CAPS Studio ist ein vollständig browserbasiertes Produktionssystem für hochwertige KI-Kinderbücher.

CAPS ist nicht der Bildgenerator, sondern der Produktionsleiter.

CAPS übernimmt:

- Projektplanung
- Szenenregie
- Prompt-Erstellung
- Bildverwaltung
- Qualitätskontrolle
- Fortschrittsverwaltung
- Projektbuch und PDF-Export
