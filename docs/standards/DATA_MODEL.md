# CAPS-Datenmodell

## Projekt

- Projekt-ID
- Titel
- Thema
- Version
- Autor
- Status
- Erstellungsdatum
- Änderungsdatum

## Seite

- Seiten-ID
- Seitennummer
- Kategorie
- Szene
- Prompts
- Bildreferenz
- QA
- Status
- Änderungsverlauf

## Speicherung

- Projektdaten: IndexedDB oder kleine Metadaten in localStorage
- Bilder: ausschließlich IndexedDB
- Projekttransfer: `.caps` oder JSON-basierter Export
- Projektdokumentation: PDF-Projektbuch
