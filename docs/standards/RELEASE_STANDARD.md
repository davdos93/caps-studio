# Stable-Freigabestandard

Eine Version darf nur als Stable veröffentlicht werden, wenn:

- die Anwendung fehlerfrei startet
- Projekte erstellt, gespeichert und geöffnet werden können
- keine Bilder verloren gehen
- Import und Export funktionieren
- der Scene Director korrekte Verteilungen erzeugt
- Prompts vollständig sind
- das PDF-Projektbuch erstellt werden kann
- Chrome Desktop getestet wurde
- Chrome iPhone getestet wurde
- keine P0- oder P1-Fehler offen sind
