# CAPS Referenzbibliothek

Diese Bibliothek enthält zwölf feste, realistische Kinderbuchfälle und die dazugehörigen Golden Masters.

Ein Golden Master speichert keine zufälligen IDs, sondern die deterministische Signatur von:

- Geschichtenentwurf und Kausalkette
- vollständigem Manuskript
- Doppelseitenrollen und Bilderbuchtexten
- Illustrationsregie und Prompts
- Konsistenz-Fundcodes

Änderungen an diesen Ergebnissen werden als Regression gemeldet und müssen bewusst geprüft werden. Die Golden Masters dürfen nur nach einer fachlichen Abnahme aktualisiert werden.
